# Minet Runner 24/7 (Railway)

Repo này dùng để chạy script Minet 24/7 trên Railway bằng Docker.

## Cài đặt
1. Fork hoặc tạo repo mới
2. Thêm các file:
   - Dockerfile
   - run.sh
3. Deploy lên Railway → Deploy từ GitHub

## Notes
- Container sẽ tự restart
- Script được chạy liên tục
- Railway Free sẽ kill nếu dùng quá CPU
